  <p align="center">Simple NestJS MongoDB application utilizing Serverless with AWS Lambda.</p>
   
## Installation

```bash
$ npm install
```

## Running the app offline

```bash
$ npm run build

$ npx serverless offline
```

## Running the app with AWS Lambda

```bash
$ npm run build

$ npx serverless deploy --stage prod 
```
